Já é possivel fazer ferias em Marte. É o uma viagem. A viagem é longa, mas é uma
experiencia unica. Pode lear varios acompanhates e realizar um grande sonho.

Mas muita atenção: são necessarios alguns itens preparatorios antes de seguir a viagem

O que é necessario para viajar?

Para viajar para Marte é necessario.

  • exames medicos gerais
  • duas semanas de treino especializado
  • Passaporte interestelar valido
  • Assinar um termo de responsabilidade

Procedimentos antes da viajar

  1. Escolher a data da viagem
  2. Efetua os exames medicos necessarios
  3. efetuar o pagamento de 60% do valor da viagem
  4. participar das duas semanas de treino especializado com exito
  5. Assinar o termo de responsabilidade
